HTML

HTML (Hypertext Markup Language) é uma liguagem de marcação baseada em hipertexto, que em resumo, são textos com hiperlinks ou links que levam de um conteúdo para outro.

Com HTML, podemos escrever documentos que serão interpretados pelo browser, pois trata-se de uma linguagem da web.

O que são tags HTML?

São elementos que identificam e organizam um conteúdo dentro de um arquivo HTML através de uma marcação (tag) entre os sinais de < e >.

Alguns exemplos conhecidos:

  • <HTML> ... </HTML> - indica que o conteúdo deve ser lido como HTML;
  • <p> ... </p> - Indica que o conteúdo é um parágrafo de texto.
  • <h1> ... </h1> - Indica que o conteúdo é um título de nível 1.
  • <h2> ... </h2> - Indica que o conteúdo é um título de nível 2.
  • <div> ... </div> - Indica que o conteúdo é uma divisão ou seção de um documento HTML, usada para agrupar elementos e aplicar estilos ou scripts a eles.

    • Uma curiosidade: antes do HTML, existiam projetos como o Xanadu, que foi desenvolvido por Ted Nelson. Ele é conhecido como o primeiro projeto de hipertexto da história.

    Tag de âncora

    A tag de âncora é representada por <a> e é utilizada para criar links em um documento HTML. Ela pode ser usada para criar links para outras páginas da web, arquivos, endereços de e-mail ou até mesmo para seções específicas dentro da mesma página.
